# 📊 FlowID 项目总览

## 🎯 项目概述

**FlowID** 是一个AI驱动的职业身份平台，让用户通过简单的多轮对话创建个性化的职业主页。平台支持多种风格、布局和展示方向，适合各类专业人士展示自己的技能、项目和经历。

## 📋 文档结构

| 文档 | 描述 | 适用人群 |
|------|------|----------|
| `README.md` | 产品需求文档，完整的功能规格 | 产品经理、开发者、投资人 |
| `DEVELOPMENT.md` | 详细的8周开发计划和技术实现 | 开发团队、技术负责人 |
| `QUICKSTART.md` | 快速开始指南，一键启动开发 | 新加入的开发者 |
| `setup.sh` | 自动化项目初始化脚本 | 所有开发者 |

## 🚀 快速开始

### 立即开始开发
```bash
# 1. 运行初始化脚本
./setup.sh

# 2. 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入API密钥

# 3. 启动开发服务器
npm run dev
```

### 详细配置
查看 `QUICKSTART.md` 获取完整的配置指南。

## 🏗 技术架构

### 核心技术栈
- **前端**: Next.js 14 + TypeScript + Tailwind CSS
- **UI组件**: Shadcn/ui + Radix UI
- **认证**: Clerk (OAuth支持)
- **数据库**: Supabase (PostgreSQL)
- **AI模型**: Claude 3.7 + OpenAI + Gemini
- **状态管理**: Zustand
- **部署**: Vercel

### 项目结构
